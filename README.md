# TA09
Problem :

Climate disasters like heatwaves, floods, and pollution spikes increase disease outbreaks.
Healthcare systems respond reactively, causing delayed funding and unprepared clinics.

Core Algorithms :

Risk Scoring Model (Weighted environmental indicators)

Disease Prediction Model (Random Forest / Logistic Regression)

Threshold-Based Funding Trigger

Supply Recommendation Engine

Technology :

Python, Flask

Scikit-learn, Pandas

SQLite / PostgreSQL

React.js / HTML + Bootstrap

Solidity (Smart Contract Prototype)
